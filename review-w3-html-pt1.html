HTML Formatting

Use html formating elements to add special meaning to the text. There are several formatting elements that are used such as:

  • <b> Defines the text as bold.
  • <strong> Defines text with strong importance.
  • <i> Defines text in an alternate voice or mood. Often used to indicate a technical term.
  • <em> Defines emphasized text. Screen readers will pronounce with emphasis.
  • <mark> Definds that the text should be marked or highlighted.
  • <small> Defines smaller text.
  • <del> Defines text that has been deleted from a document.
  • <ins> Defines text that has been inserted into a document.
  • <sub> Defines subscript text.The text appears half a char below the normal line. Subscript can be used for formulas line H2O.
  • <sup> Defines superscript text. Superscript text appears half a char above the normal line. Superscript can be used for foot notes like WWW[1]

HTML CSS

CSS stands for Cascading Style Sheets. CSS is used to format the layout of your web page. With CSS you can control how your web page looks. CSS has a vast amount of attributes to make your web page look pretty!

There are three different ways to add CSS to your HTML document.

  • Inline - by using the sytle attribute inside your element as seen above.
  • Internal - by using the <style> element in the <head> section.
  • External - by using a <link> element to link to an external CSS file.

The most common way to add CSS is to keep the styles in external CSS files.

HTML Links

Html links are hyperlinks to another document. To add a link the <a> element is used with the href attribute. For example Free Coding Bootcamp will take you to the Free Coding Bootcamp web site.

You can use the <target> attribute to choose how the document opens such as:

  • _self - Is default. Opens the document in the same window or tab.
  • _blank - Opens the document in a new window or tab.
  • _parent - Opens the document in the parent frames.
  • _top - Opens the document in the full body of the window.

HTML Images

You can insert images into your document by using the <img> element. This element does not have a closing tag. The <img> needs to have the src attribute to point to the location of the image and the alt attribute that specifies an alternate text for the image. If the link is broken the alternate text will be shown. If the user is using a screen reader the alternate text will be read. CSS should be used to position and size the image.

HTML Tables

HTML tables are used to arrage data into rows and columns. The <table> element is used to define a table. There many different tags that need to set be to configure the table. CSS can be used to style the table. Here are the HTML table tags:

Tag Name Description
<table> Defines a table
<th> Defines a header in the table
<trth> Defines a row in the table
<td> Defines a cell in the table
<caption> Defines a table caption
<colgroup> Specifies a group of one or more columns in a table for formatting
<col> Specifies column properties for each column within a <colgroup> element
<thead> Groups the header content in a table
<tbody> Groups the body content in a table
<tfoot> Groups the footer in the content table
